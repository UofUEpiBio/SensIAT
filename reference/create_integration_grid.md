# Create integration grid

Create integration grid

## Usage

``` r
create_integration_grid(tmin, tmax, n_grid = 100, obs_times = NULL)
```

## Arguments

- tmin:

  Lower bound

- tmax:

  Upper bound

- n_grid:

  Number of grid points

- obs_times:

  Optional vector of observation times to include

## Value

Sorted numeric vector of grid points
