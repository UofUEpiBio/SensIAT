# Sample parametric coefficients from a fitted model using asymptotic multivariate normal distribution.

Sample parametric coefficients from a fitted model using asymptotic
multivariate normal distribution.

## Usage

``` r
sample_parametric_coeffs(model)
```

## Arguments

- model:

  A fitted model with [`coef()`](https://rdrr.io/r/stats/coef.html) and
  [`vcov()`](https://rdrr.io/r/stats/vcov.html) methods

## Value

Named numeric vector of sampled coefficients
